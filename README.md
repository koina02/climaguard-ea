# ClimaGuard EA

ClimaGuard EA is a lightweight MVP climate anomaly detection dashboard for the East African region. Built for mobile-first interaction, it offers users live climate status updates, and can be extended to integrate weather APIs, anomaly alerts, and regional forecasting.

## Features
- Minimalist UI for rapid feedback
- Designed for East African jurisdictions
- Mobile-compatible MVP layout
- Easily extendable with APIs and backend

## Roadmap
